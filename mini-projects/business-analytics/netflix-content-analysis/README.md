# Netflix Content Analysis

This mini project explores Netflix movies and TV shows to uncover insights about content types, genres, countries, release trends, and ratings using exploratory data analysis and visualization techniques.

## Project Scope

The analysis focuses on:
- Understanding the distribution of movies vs TV shows
- Identifying popular genres and producing countries
- Exploring release trends over time
- Analyzing ratings and director level patterns

## Key Steps

- Data loading and basic cleaning
- Feature exploration and aggregation
- Visual analysis using static and interactive plots
- Time based trend analysis (year and month)
